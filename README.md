# ShopEasy – E-Commerce Web Application

ShopEasy is a full-stack e-commerce web application designed to demonstrate the functionality of a modern online shopping platform. The project is developed using HTML, CSS, and JavaScript for the frontend interface, while Python Flask is used to handle the backend server operations and MySQL is used as the database for storing application data.

The platform allows users to browse multiple products displayed in a responsive grid layout similar to popular e-commerce websites. Each product contains an image, name, price, and an option to add the item to the shopping cart. The shopping cart system enables users to manage selected items, remove products, and view the total cost before placing an order.

The backend server handles user authentication, product retrieval, and order storage using REST-based APIs. All important information such as user accounts, product details, and order records are securely stored in the database. The communication between frontend and backend makes the system dynamic and interactive.

## Technologies Used

* HTML
* CSS
* JavaScript
* Python (Flask)
* MySQL

## Features

* User Login System
* Product Listing Page
* Shopping Cart System
* Order Placement
* Backend Database Integration

This project demonstrates the basic architecture of a full-stack e-commerce system and serves as a learning project for web development.
